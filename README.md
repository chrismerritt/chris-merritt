# 🎉 Getting Started

1. 📦 Install dependencies.
   ```sh
   npm i
   ```
2. 🚧 Build the site for development and run a development server.
   ```sh
   npm start
   ```
3. 🚢 Build the site for production deployment

   ```sh
   npm run generate
   ```
